# 🎮 Game Diamond Top Up

Website top up game berbasis MERN Stack yang dilengkapi dengan dashboard admin, integrasi Midtrans, validasi nickname game, dan sistem notifikasi Telegram.

## ✨ Features

- Login Admin (JWT Authentication)
- Dashboard Admin
- CRUD Product
- Checkout menggunakan Midtrans
- Tracking pesanan
- Validasi nickname game
- Contact Form
- Telegram Notification
- Responsive Design

## 🛠 Tech Stack

**Frontend**
- React
- Tailwind CSS
- React Router

**Backend**
- Node.js
- Express.js
- MongoDB
- JWT
- bcrypt
- express-validator

**Integrations**
- Midtrans
- API Games
- Telegram Bot API
